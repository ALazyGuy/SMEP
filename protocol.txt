Protocol(Smart Messages Exchange Protocol):
1) 1 byte - device id
2) 1 byte - message type
3) ((1 byte - field size = n) + (n bytes - field data)) * (m - number of fields for current messge type)

Total size: 2 + (1 + n) * m bytes

Message types:
1) 0x0 - initial scan(SRV)
2) 0x1 - online(CL)
3) 0x2 - accepted(SRV)
4) 0x3 - accepted(CL)
5) 0x4 - update(SRV)
6) 0x5 - update(CL)

Handshake:

1) Server does scan for all devices connected to specific network by sending initial scan message(0x0) to all devices
2) Every device which is available in current network respond with both accepted(0x3) and online(0x1) messages
3) Server saves each device to local storage with all provided data and sends accepted(0x2) message to all connected devices
